Комплексное функциональное тестирование популярного демо-приложения **Swag Labs** — интернет-магазина с разными типами пользователей.

**Цель проекта:**
- Практика ручного тестирования
- Составление тестовой документации
- Выявление и документирование багов
- Подготовка портфолио для Junior QA позиции
